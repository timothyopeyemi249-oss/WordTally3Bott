# WordTally3Bot 🤖

A Telegram bot that counts words, characters, sentences, and paragraphs in any text.

## Features ✨
- Count words in text messages
- Analyze .txt and .md files
- Character, sentence, and paragraph statistics
- Inline buttons for easy navigation
- Deployed on Railway

## Commands 🛠️
- `/start` - Welcome message
- `/help` - Help menu
- `/stats` - Your usage statistics

## Deployment 🚀
1. Clone this repository
2. Create a `.env` file with your `TELEGRAM_BOT_TOKEN`
3. Deploy on Railway

## Tech Stack 💻
- Python 3.11+
- python-telegram-bot
- Railway
